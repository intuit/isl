# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.2-SNAPSHOT] - 2026-03-20

- Added YAML input/output support in the playground (680e9dd)
- Added support for object-style `if` expressions (f6cd805)
- Added language/runtime optimizations for `group.by`, spread assignment, map+filter, and interpolation (eb19070)
- Fixed incorrect timezone calculation and null crash in `DatePagination` (d1b3bd1)
- Improved runtime error reporting for crashes inside ISL (d1b3bd1)
- Improved command-line usage, validations, and test framework capabilities (f19cc5e, 889ad6f, f23753b, 1cf7328, d09f049)
- Improved plugin testing, validations, code actions, and imported extension support (16d9430, 42c231b, 40b2128, db73d7d, 59486c1, 532b549, e9676f6, 9a654c3, 1e30363)
- Added plugin packaging and GitHub CI/CD/marketplace workflows (31e7e98, 367d94d)
- Added build/release improvements including Maven publishing reliability, runtime build steps, and deployment updates (c5f6efb, c8d2644, bd4ad8d, 529379f, 35d1f44)
